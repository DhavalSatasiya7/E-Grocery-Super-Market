# 🛒 E-Grocery Super Market

## 📖 Project Summary

E-Grocery Super Market is a modern online grocery store interface developed using HTML and CSS. The project is designed to provide users with a simple and attractive platform for browsing grocery products, viewing prices, and exploring product categories.

The website features a clean layout with a header section, product search functionality, sorting options, product showcase cards, and a footer containing contact information. It demonstrates the fundamentals of front-end web development and responsive user interface design.

The product catalog includes a variety of grocery items such as snacks, chocolates, coffee, dairy products, cookies, and daily essentials. Each product is displayed with an image, description, price, and an "Add to Cart" button to simulate an e-commerce shopping experience.

## ✨ Key Features

* Attractive Grocery Store Landing Page
* Product Search Bar
* Product Sorting Options
* Product Cards with Images and Prices
* Responsive Layout Design
* Header and Footer Sections
* User-Friendly Interface
